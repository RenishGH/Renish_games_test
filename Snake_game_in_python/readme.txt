# Snake Game in Python

A classic Snake game implemented in Python using Pygame. Move the snake to collect letters forming the name **"Renish Mirani"**. Each letter eaten increases the snakeâ€™s speed and changes the background color, making the game more challenging and visually engaging.

## Features

- Smooth snake movement with wrap-around screen edges (snake appears from the opposite side)
- Collect letters one by one to spell **"Renish Mirani"**
- Increasing snake speed after each letter collected
- Dynamic background color change after each letter eaten
- Start menu with instructions
- Real-time scoreboard displaying progress
- Game Over screen with option to restart or quit
- Responsive controls using arrow keys
- Sound effects for eating letters and game events

## Requirements

- Python 3.x
- Pygame (`pip install pygame`)

## How to Play

1. Run the game with `python snake_game.py`
2. Use arrow keys to move the snake.
3. Collect letters in the correct order to complete the name.
4. Avoid running into yourself.
5. The snake wraps around the edges instead of dying.
6. After each letter, the speed increases and background color changes.
7. Press `ESC` to quit anytime.
8. After game over, press `R` to restart or `Q` to quit.
