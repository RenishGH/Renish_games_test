# Tic Tac Toe Game

A simple Tic Tac Toe game built with Python and Tkinter that supports both one-player (vs computer) and two-player modes.

---

## Features

- Choose to play **One Player (vs Computer)** or **Two Players**.
- Computer opponent plays as "O" with basic random moves.
- Graphical user interface using Tkinter.
- Game detects wins, ties, and prompts to restart or quit.
- Simple and clean UI for an easy gaming experience.

---

## Requirements

- Python 3.x
- Tkinter (usually included with standard Python installation)

---

## How to Run

1. Save the script file (e.g., `tic_tac_toe.py`).
2. Open a terminal or command prompt.
3. Run the script using:

   ```bash
   python tic_tac_toe.py
